## 具体用法
### 邮箱设置
启用 smtp/imap服务或pop3/imap服务  
这种服务很早就已经存在了，可以帮助我们将邮件拉取到本地或在本地发送邮件  
服务要在邮箱中设置，比如163：

因为我的服务已经开启，用文字说明接下来的步骤  
大概是要你先进行邮箱验证，然后会给你一个密码，这个密码就是你的服务密码，一定要记住，**只会展示一次！**  
拓展一下，如果您以后要配置本地客户端，需要使用这个密码，你可以选择再创建一个密码，或使用原来的密码，因此这个密码一定要保存好。
